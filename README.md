# MethodDka — 高速・高精度な因数分解 Web アプリ

MethodDka は、HTML と JavaScript だけで動作する **軽量・高速・高精度な因数分解ツール**です。DKA法とホーナー法を統合した世界基準の高精度数値解析リファレンス（v2.0.0）です。

## 📦 オフライン利用（ZIP / tar.gz）

GitHub の最新リリースからソースコード一式を入手できます。インストール作業は不要で、展開して `MethodDka.html` をブラウザで開くだけで動作します。
