📘 MethodDka とは

MethodDka は、因数分解 および 高次代数方程式の全解（複素数を含む）を求めるプログラムです。

  • HTML / JavaScript で動作
  • オフライン版はダウンロードして使用可能
  • オンライン版はブラウザだけで動作
  • 高精度版は約 50 桁の計算に対応

🔢 MethodDka の特徴

  • 最大 1000 次 の多項式に対応
  • 複素数の全解 を求められる
  • 初期値不要 で収束（DKA 法）
  • ブラウザで動作（HTML / JavaScript）
  • 高精度版は 50 桁 の計算に対応
  • Windows 版は MSI / APPX によるインストール形式

🧮 使い方（概要）

  1. 次数を入力
  2. 係数を入力
  3. 「計算開始」ボタンを押す
  4. 結果が画面に表示されます

⚠ 注意事項

本プログラムは試作評価版です。計算結果の正確性を保証するものではありません。自己責任で使用してください。
