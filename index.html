📘 MethodDka とは

因数分解や高次代数方程式を解くためのプログラムです。高精度バージョン（50桁）にも対応しています。

🧮 概要

  • MethodDka.html（JavaScript）
  • 対応 OS：Windows 11 Home 25H2、Android 15
  • MIT ライセンス
  • 試作評価版：Ver 6.00
  • 作者：小泉 嘉章
  • メール：ja142397@s6.dion.ne.jp

📐 数学的背景

  • 1次方程式 → 中学1年
  • 2次方程式 → 中学3年
  • 3次以上 → カルダノの公式・数値解析
  • 4次以上の一般方程式 → 数値解析が必須

⭐ MethodDka の特徴

  • 実数係数の高次（最大 1000次）多項式に対応
  • 複素数を含むすべての解を求められる
  • 初期値なしで収束（DKA法：Durand–Kerner–Aberth 法）
  • ブラウザだけで動作（Edge / Chrome）
  • Android スマホでも動作
  • 高精度版は約50桁の有効数字

🛠 使い方

  1. 最高次数を入力
  2. 次数の高い順に係数を入力
  3. 係数が存在しない場合は必ず 0 を入力
  4. 計算開始ボタンを押す
  5. 全ての解が表示される
