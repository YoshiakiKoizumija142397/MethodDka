📦 Download ZIP (v1.0.0)

ZIP を展開し、MethodDka.html を開くだけで利用できます。

🧮 機能

  • 大きな整数の因数分解
  • 高速アルゴリズムによる計算
  • 入力値のバリデーション
  • 結果の即時表示
  • オフライン動作（ZIP版）

📁 ファイル構成

MethodDka/
├── MethodDka.html   # メイン画面
├── style.css        # UI スタイル
├── script.js        # 因数分解ロジック
└── assets/          # 画像・追加リソース（必要に応じて）

🛠 使い方

  1. Web 版または ZIP 版を開く
  2. 因数分解したい整数を入力
  3. 「計算」ボタンを押す
  4. 結果が即時に表示されます
