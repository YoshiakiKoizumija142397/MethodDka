📘 MethodDka とは

因数分解したり、高次代数方程式を解くプログラム。高精度バージョン（50桁）にも対応。

概要

  • MethodDka.html（JavaScript）
  • 対応OS：Windows 11 Home 25H2、Android 15
  • MIT ライセンス
  • 試作評価版バージョン：6.00
  • 作者：小泉 嘉章
  • メール：ja142397@s6.dion.ne.jp

数学的背景

  • 1次方程式 → 中学1年
  • 2次方程式 → 中学3年
  • 3次以上 → カルダノの公式・数値解析（ラグランジュ法・ニュートン法）
  • 4次以上の一般方程式 → 数値解析が必須

MethodDka の特徴

  • 実数係数の高次（最大 1000次）多項式に対応
  • 複素数を含む すべての解 を求められる
  • 初期値なしで収束（DKA法：Durand–Kerner–Aberth 法）
  • ブラウザだけで動作（Edge / Chrome）
  • Android スマホでも動作
  • 高精度版は 約50桁の有効数字

使い方

  1. 最高次数を入力
  2. 表示された入力ボックスに、次数の高い順から係数を入力
  3. 次数が存在しない場合は必ず 0 を入力（省略不可）
  4. 係数が +1 の場合は「1」、-1 の場合は「-1」を入力
  5. 計算開始ボタン を押すと、全ての解が表示される

DKA 法が収束しない場合の要因

  • 初期値の選定が不適切
  • 重根や近接する根が存在する
  • 係数のスケールが極端に異なる
  • 反復回数や許容誤差の設定が不十分
  • 複素数演算での丸め誤差の蓄積
