MethodDka v2.0.0

DKA法とホーナー法を統合した、世界基準の高精度数値解析リファレンスです。

インストール不要・全OS対応。 ブラウザから即座に因数分解を実行できます。

今すぐ計算する（世界基準版）

🌐 Webアプリとしての特徴

  • インストール不要: ブラウザ（Edge/Chrome 推奨）で開くだけで動作します。
  • 全OS対応: Windows, Mac, Linux, Chromebook, ChromeOS Flex に対応。
  • アプリ化対応: ブラウザ機能で「アプリとしてインストール」すれば、オフラインでも利用可能です。
  • 日本のスマホ新法に対応: Google/Apple/Microsoft 各ストアのインストールは一切不要です。

📦 オフライン利用

GitHub リポジトリから ZIP をダウンロードし、ブラウザで MethodDka.html を開くことでオフライン動作します。

※iOS の場合も Edge/Chrome ブラウザ経由で動作可能です。
