📘 MethodDka とは

因数分解したり、高次代数方程式を解くプログラムです。高精度バージョン（50桁）にも対応しています。

対応 OS（オンライン版）

Windows 11 / Android / iOS / Mac / Amazon デバイス / Linux / Chromebook / ChromeOS Flex

概要（JavaScript / HTML 版）

オンライン版はブラウザのみで動作し、追加インストール不要です。

🔢 MethodDka の特徴

  • 最大 1000 次の多項式に対応
  • 複素数の全解を求められる
  • 初期値なしで収束（DKA 法）
  • ブラウザで動作（HTML / JavaScript）
  • 高精度版は約 50 桁の計算に対応

🧮 使い方（概要）

  1. 次数を入力
  2. 係数を入力
  3. 「計算開始」ボタンを押す
  4. 結果が画面に表示されます

本プログラムは試作評価版です。自己責任で使用してください。
